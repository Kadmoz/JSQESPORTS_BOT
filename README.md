# JSQESPORTS_BOT
Bot para manejo de ligas de JSQ ESPORTS en discord


🏎️ Bot de Simracing - Guía de Uso
El bot ya está activo 24/7. Aquí los comandos principales:

⚙️ CONFIGURACIÓN INICIAL (Solo admins):

!canal_recordatorios
(Escribir esto en el canal donde quieren recibir notificaciones automáticas 1 hora antes de cada carrera)

📝 ACTUALIZAR INFO DE LIGAS:
!actualizar Liga Canalla Prime | hora: 20:30 | admin: Juan | stream: twitch.tv/canal


📅 COMANDOS PARA TODOS:

!calendario o !cal - Ver calendario completo de la semana
!hoy - Ver carreras de hoy
!proxima o !pc - Ver próxima carrera
!info Liga Canalla Prime - Ver detalles de una liga específica
!ayuda_bot - Ver todos los comandos

🔧 COMANDOS DE ADMIN:

!nuevo_torneo Sábado Nombre de la Liga - Añadir nueva liga
!eliminar_torneo Nombre - Eliminar una liga
!actualizar Nombre Liga | hora: XX:XX | admin: Nombre | stream: link - Actualizar info
